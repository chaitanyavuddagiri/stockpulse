# StockPulse

AI-Powered Stock Research Platform build with React 19 and .NET 10

## Tech Stack

- **Frontend**:  React 19, Typescript, Vite, Tailwindcss, shadcn/ui
- **Backend**:  .Net 10, MongoDB, Redis
- **AI**: Anthropic Claude API
- **Deployment**: Docker, Azure
